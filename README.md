# Shaavik Legal Website

Single-page static website for Shaavik Legal — a business and legal support consultancy based in London.

## Stack
Plain HTML, CSS, and JavaScript. No build step required.

## Structure
- `index.html` — full site (markup, styles, scripts inline)
- `assets/` — logo and section imagery
